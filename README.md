# embedded-core-lab

Hands-on embedded systems engineering lab focused on low-level programming, HAL/driver development, bootloaders, RTOS, and Linux kernel internals.

## Roadmap

- Memory model
- Bit manipulation
- Register programming
- HAL abstraction
- Drivers
- Linker / startup
- Bootloader
- RTOS internals
- Linux kernel modules
- Embedded architecture
